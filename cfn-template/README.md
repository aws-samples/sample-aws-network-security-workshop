# CloudFormation Template

This directory contains the CloudFormation template used to deploy the workshop environment for the [AWS Network Security Workshop](https://catalog.us-east-1.prod.workshops.aws/workshops/cbfa9f18-1175-4916-b7e7-e33dbbca9f9d).

## Template File

- **template.yaml**: CloudFormation template that deploys the complete workshop infrastructure

## Deployment

For deployment instructions, please refer to the main [README.md](../README.md) in the repository root.

## Region

This template is designed to be deployed in **US East (N. Virginia) - us-east-1**.
